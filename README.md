## ✨ Features

- 💬 Real-time AI chat powered by Claude (Anthropic)
- 🎨 Stunning dark UI with animated grid background
- ⚡ Quick-reply buttons for common questions
- 📱 Fully responsive — works on mobile & desktop
- 🔐 API key entry on launch (no backend needed)
- 🌊 Typing indicator with smooth animations

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/weshytonnie-hub/ai-receptionist.git
cd ai-receptionist
```

### 2. Open in browser
Just open `index.html` in your browser — no install needed!

```bash
open index.html
# or double-click the file
```

### 3. Get an API Key
- Go to [console.anthropic.com](https://console.anthropic.com)
- Sign up for a free account
- Generate an API key
- Paste it into the ARIA launch screen

## 🔧 Customization

Open `index.html` and edit the `SYSTEM_PROMPT` variable to personalize ARIA:

```javascript
const SYSTEM_PROMPT = `You are ARIA, receptionist for [YOUR NAME]...
- Your skills: HTML, CSS, Java, C++
- Location: Nairobi, Kenya
- Contact: weshytonnie@gmail.com
`;
```
